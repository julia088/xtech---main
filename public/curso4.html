Responda as perguntas a seguir:


Python é uma linguagem de programação de alto nível, usada em desenvolvimento web, análise de dados, inteligência artificial, automação e mais.
Python é uma ferramenta de design gráfico usada para criar interfaces visuais.
Python é um sistema operacional utilizado para gerenciar servidores.


Você pode declarar uma variável simplesmente atribuindo um valor a ela, por exemplo: meu_numero = 10.
Para declarar uma variável, você precisa usar a palavra-chave var.
Você deve declarar uma variável dentro de uma função para que ela seja reconhecida.


Listas são coleções ordenadas e mutáveis de elementos, enquanto tuplas são coleções ordenadas e imutáveis.
Listas são tipos de dados que podem armazenar apenas números, enquanto tuplas armazenam textos.
Listas e tuplas são a mesma coisa, mas as tuplas têm um limite de tamanho.


Você pode criar uma função usando a palavra-chave def.
As funções são criadas automaticamente quando um programa Python é executado.
Para criar uma função, você precisa usar a palavra-chave function.


Bibliotecas são coleções de módulos e funções que você pode usar para adicionar funcionalidades ao seu código. Para usar uma biblioteca, você deve importá-la, como em import math.
Bibliotecas são apenas arquivos de texto que você pode abrir com qualquer editor.
Bibliotecas são serviços de nuvem que armazenam seus dados.
