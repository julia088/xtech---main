JavaScript é uma linguagem de programação que permite adicionar interatividade e dinamismo a páginas da web, sendo executada no lado do cliente (navegador).
JavaScript é uma linguagem de marcação usada apenas para criar links em páginas web.
JavaScript é um sistema de gerenciamento de banco de dados para aplicações web.


var tem escopo de função e pode ser redeclarado. let tem escopo de bloco e não pode ser redeclarado. const também tem escopo de bloco, mas é usado para declarar constantes.
var e let têm o mesmo escopo, enquanto const pode ser redeclarado em qualquer lugar.
var é mais seguro que let e const, pois pode ser usado globalmente.


Um evento é uma ação que ocorre em um elemento da página, como um clique ou um movimento do mouse. Você pode manipulá-lo usando event listeners, como element.addEventListener('click', function).
Um evento é uma função que se executa automaticamente ao carregar a página.
Um evento é um erro que ocorre quando o JavaScript não é interpretado corretamente.


Uma função pode ser definida usando a palavra-chave function.
Funções são definidas automaticamente quando a página é carregada.
Funções são apenas uma forma de adicionar comentários ao código JavaScript.


O DOM (Document Object Model) é uma representação em árvore de um documento HTML. JavaScript pode interagir com o DOM para modificar a estrutura, estilo e conteúdo da página.
O DOM é uma linguagem de programação utilizada apenas para estilizar páginas web.
O DOM é um tipo de banco de dados usado para armazenar informações de usuários.
