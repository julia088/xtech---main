Responda as perguntas a seguir:


PHP é uma linguagem de programação de código aberto amplamente usada para desenvolvimento web, especialmente para criar páginas dinâmicas e interativas.
PHP é um software de edição de imagens.
PHP é uma linguagem de marcação para criar documentos.


Você pode declarar uma variável em PHP usando o sinal de cifrão ($).
Variáveis em PHP são declaradas sem um símbolo.
Você deve usar a palavra-chave "var" para declarar uma variável.


Arrays são coleções de valores que podem ser acessados por chaves ou índices.
Arrays são funções que retornam valores.
Arrays são arquivos que armazenam dados.


Sessões permitem armazenar dados do usuário em várias páginas. Você inicia uma sessão usando session_start() e pode armazenar dados em `$_SESSION`.
Sessões em PHP são utilizadas para gerenciar cookies.
Sessões são usadas apenas para autenticação de usuários.


Funções são blocos de código reutilizáveis que executam uma tarefa específica.
Funções são variáveis que armazenam dados temporariamente.
Funções são instruções que não podem ser reutilizadas.
