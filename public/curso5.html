Ruby é uma linguagem de programação dinâmica, usada principalmente no desenvolvimento web, especialmente com o framework Ruby on Rails.
Ruby é uma linguagem de script usada apenas para automação de tarefas administrativas.
Ruby é uma ferramenta de design gráfico utilizada para criar interfaces.


Você pode criar uma variável simplesmente atribuindo um valor a ela.
Para declarar uma variável, você precisa usar a palavra-chave let.
Variáveis em Ruby precisam ser declaradas dentro de classes.


Classes são moldes que definem propriedades e comportamentos de objetos. Objetos são instâncias de classes.
Classes e objetos são sinônimos e podem ser usados de forma intercambiável.
Classes são apenas funções que podem ser chamadas diretamente, enquanto objetos são valores fixos.


A interpolação de strings permite inserir valores de variáveis dentro de uma string.
A interpolação de strings é o processo de converter uma string em um número.
A interpolação de strings é uma técnica para dividir strings em múltiplas partes.


Gems são bibliotecas ou pacotes que adicionam funcionalidades a aplicações Ruby. Você pode instalá-las usando o comando gem install nome_da_gem.
Gems são tipos de variáveis que armazenam dados temporários em Ruby.
Gems são apenas nomes de arquivos de texto usados em Ruby.
