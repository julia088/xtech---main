Responda as perguntas a seguir:


CSS significa "Cascading Style Sheets" e é usado para descrever a apresentação e o layout de um documento HTML.
CSS significa "Creative Style Sheets" e é usado apenas para criar links.
CSS significa "Cascading Style Scripts" e é utilizado para inserir imagens.


Você pode aplicar estilos CSS a um elemento usando seletores.
Você pode aplicar estilos CSS apenas a elementos com IDs.
Você pode aplicar estilos CSS apenas por meio de atributos HTML.


Seletores são maneiras de aplicar cores a elementos. Os tipos mais comuns incluem cores hexadecimais e RGB.
Seletores são padrões usados para selecionar elementos HTML a serem estilizados. Os tipos mais comuns incluem seletores de tipo (ex: div), classe (ex: .classe) e ID (ex: #id).
Seletores são scripts que ajudam a executar funções em JavaScript.


O modelo de caixa representa cada elemento como uma caixa retangular composta por margens, bordas, preenchimento (padding) e o conteúdo em si.
O modelo de caixa é uma técnica para aplicar cores de fundo aos elementos.
O modelo de caixa define a posição de um elemento na página.


Uma folha de estilo externa é um arquivo CSS separado que pode ser linkado a múltiplas páginas HTML. Já a folha de estilo interna é definida dentro da tag "style" em um documento HTML.
Uma folha de estilo externa é uma folha de cálculo para dados. A interna é apenas para cores.
Folhas de estilo externas são usadas apenas para impressão, enquanto internas são para visualização na tela.
